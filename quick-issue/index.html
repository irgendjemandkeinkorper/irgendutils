How to create the token
  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: the repos you'll file issues in (or All).
  3. Permissions → Issues: Read and write, Contents: Read and write (for attachment uploads), Metadata: Read.
  4. Paste it above. Stored only in this browser's localStorage — never sent anywhere but api.github.com.
